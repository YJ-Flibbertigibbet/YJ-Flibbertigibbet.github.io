Chapter 1

如何理解一致收敛与极限换序

设{$f_n(x)$}在$(a,b)$上一致收敛到$f(x)$，设对每个$n\geq 1$都有:$f_n(a^+)=\lim_{x\rightarrow a^+}f(x)=\lim_{x\rightarrow a^+}\lim_{n\rightarrow \infty}f_n(x)$=$\lim_{n\rightarrow \infty}\lim_{x\rightarrow a^+}f_n(x)$

事实上，我们可以利用n和x张成一个不完备的二维空间：不妨设$F(x_0,\frac{1}{n})=f_n(x_0)$,从而可以类比二维层面上的一致收敛

概率论整理

多项式系数

记 $\sum_{i=1}^rn_i=n$ 考虑把 $n$ 个元素分为 $r$ 组,问有多少种分法，定义$(\overset{n}{n_1,n_2…,n_r})=\frac{n!}{n_1!n_2!…n_r!}$

值得一提的是这里的 $(\overset{n}{n_1,n_2…,n_r})$ 考虑了顺序，分成了即从n个事件中分出$r$份，这里的$n_1,n_2…n_r$地位并不相同，从 $1$ 到 $r$ 具有一定顺序

方程整数解的个数

值得注意的是非整数解和正整数解的求解区别

只讨论非负整数解的个数：$\sum_{i=1}^rx_i=x$ ,讨论 $(x_1.x_2…x_r)$ 向量非负解的个数

注意到在各个事件之间最多可以插空r次，从 $1$ 到 $r$ 次插空，第 $i$ 次插空时，不妨记插空在 $x_j$ 之后，则第 $i+1$ 次插空有 $x-j+1$ 个选择， 即每次插空会在原有基础上多出 $1$ 次插空的位置，因为允许两次插空在相同位置，从而可以转换为对 $x+r$ 个事件做 $r-1$ 次分割得正整数解

备注：这里做 $r-1$ 次分割但要求多出 $r$ 个事件是由于每个事件的分类都可能在0处，即为保证至少每个分类得到的都有 $1$ 个事件

容斥恒等式

$P(E_1\cup E_2 \cup …\cup E_n)=\sum_{r=1}^n(-1)^{r+1}\sum_{i_1<…<i_r}P(E_{i_1} \cup …\cup E_{i_r})$

直观解释：$\sum_{i=0}^m (\overset{m}{i}) (-1)^i=0$ , 即对任意事件只被计算一次。被包含于 $m$ 个事件中，穷尽所有子集进行运算后通过加减使得每个事件都只计算一次。而这在容斥恒等式中是对任意包含于 $E_1\cup E_2 \cup …\cup E_n$ 的事件都成立

不等式推广

$\begin{cases}
P(\cup_{i=1}^nE_i)\leq \sum_{i=1}^nP(E_i)\\
P(\cup_{i=1}^nE_i)\geq \sum_{i=1}^nP(E_i)-\sum_{j<i}P(E_iE_j)\\
P(\cup_{i=1}^nE_i)\leq \sum_{i=1}^nP(E_i)-\sum_{j<i}P(E_iE_j)+\sum_{k<i<j}P(E_kE_iE_j)\\
……
\end{cases}$

注意到：$P(\cup_{i=1}^nE_i)=P(E_1)+\sum_{i=2}^nP(E_1^cE_2^c…E_{i-1}^cE_i)$ , 即每次对集合做切割，记 $B_i=(E_i\cup_{j<i}E_j)$,而 $P(E_i)=P(B_iE_i)+P(B_i^cE_i)$

对$P(E_i)=P(B_iE_i)+P(B_i^cE_i)$两边从 $2$到 $n$ 求和，经过替换可以得到 $P(\cup_{i=1}^nE_i)=\sum_1^nP(E_i)-\sum_2^nP(B_iE_i)$,于是第一个不等式得证

同理其他不等式得证

配对问题(帽子)

$N$ 个人分配 $N$ 顶帽子，讨论所有人都没有拿到自己帽子的概率

记 $E_i$ 为第 $i$ 个人拿到自己帽子的事件，从而利用容斥定理分别计算至少有1个人分配到自己帽子的概率 $P(\cup_1^nE_i)=\sum_{n=1}^N(-1)^{n+1}\sum_{i_1<…<i_n}P(E_{i_1}\cup…\cup E_{i_n})$

$\sum_{i_1<…<i_n}P(E_{i_1}\cup…\cup E_{i_n})=(\overset{n}{N})\frac{(N-n)!}{N!}=\frac1{n!}$,从而利用容斥不等式知： $P(\cup_1^nE_i)=\sum_1^N(-1)^{n+1}\frac1{n!}$

最后做补集可得概率 $\sum_0^N(-1)^{n}\frac1{n!}$ ，当 $N$ 足够大时知概率接近 $e^{-1}$

游程 （contributor：YJ & $\phi’(x)$)

在一组数据中连续出现的相同数值的长度(一般为二分形式)

分析与例题

假设有一段成功率为$p$的序列，求连续$n$次的成功在连续$m$次失败之前的概率。

记：$E$为事件“连续$n$次成功在连续$m$次失败之前” , $F$为事件“第$2$到$n$次都成功”, $H_1$为事件“第$1$次成功”。

本题求解不是我们的重点，我们重点分析课本中出现的一个式子：

事件$HF^c$意味着前$k(k<n)$次是连续成功的，而第$k+1$次是失败的，我们要考虑$n$次成功在$m$次连续失败之前的事件，因为第$k+1$次实验是失败的，所以前$k$次成功对$E$ 没有任何影响，无论$k$等于多少，也要从第$k+1$次重新计算连续成功的次数，和实验只做了$1$次但失败了是一个效果。

或者用数学的记号来说明：记 $H_i(i = 1,2,\cdots ,n)$ 为事件“第 $i$ 次试验成功”。

$B_i = H_1H_2\cdots H_{i - 1}H_i^c(i =1,2,\cdots,n)$为事件“前 $i -1$ 次试验成功，第 $i$ 次试验失败。满足$\displaystyle \bigcup_{i = 2}^nB_i = F^cH_1,\ \bigcap_{i = 1}^nB_i = \varnothing$

则式子转化为：

$
P(E|\bigcup_{i = 2}^nB_i) = P(E|B_1)
$

现在我们把问题抽象出来，如果我们记$H$为事件“前$n$次试验出现失败”，记$A = F^cH_1$，$B = H_1^c$，那么有$A\cup B = H,A\cap B = \varnothing$。则此时有$P(E|A) = P(E|B) = P(E|H)$。对于两个事件成立这样的等式，那么对于多个事件，这样的等式是否仍然成立？

推广1：
设$P(E|F_i) = p$，$\displaystyle \bigcup_{i = 1}^nF_i = F,\bigcap_{i = 1}^nF_i = \varnothing$，则：
$
P(E|F_i) = P(E|\bigcup_{i = 1}^ n F_i) = p
$

证明：
$
P(E|\bigcup_{i = 1}^nF_i) = \frac {P(E\bigcup_{i = 1}^nF_i)}{\sum_{i = 1}^nP(F_i)} = \frac {\sum_{i = 1}^nP(EF_i)}{\sum_{i = 1}^nP(F_i)} = \frac {\sum_{i = 1}^nP(E|F_i)P(F_i)}{\sum_{i = 1}^nP(F_i)} = p
$
在得到这样的结论，我们自然会设想，假如$P(E|F_i) = p_i(i = 1,2,\cdots,n)$，$\displaystyle \bigcup_{i = 1}^nF_i = F,\bigcap_{i = 1}^nF_i = \varnothing$，会有类似的结论吗？

推广2：
设$P(E|F_i) = p_i$，$\displaystyle \bigcup_{i = 1}^nF_i = F,\bigcap_{i = 1}^nF_i = \varnothing$，则：
$
P(E|\bigcup_{i = 1}^ n F_i) = \sum_{i = 1}^n\frac {P(F_i)}{P(F)}p_i
$

证明：
$
P(E|\bigcup_{i = 1}^nF_i) = \frac {P(E\bigcup_{i = 1}^nF_i)}{\sum_{i = 1}^nP(F_i)} = \frac {\sum_{i = 1}^nP(EF_i)}{\sum_{i = 1}^nP(F_i)} = \frac {\sum_{i = 1}^nP(E|F_i)P(F_i)}{\sum_{i = 1}^nP(F_i)} = \sum_{i = 1}^n\frac {P(F_i)}{P(F)}p_i
$

直观上，推广2就是对条件概率的加权平均，推广1是推广2的一种特例。
